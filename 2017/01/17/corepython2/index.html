正则表达式与python语言

默认所有程序已经导入 Re 模块 
import re

python 中的 re 模块支持更强大而且更通用的 Perl 风格(Perl 5 风格)的正则表达式，该模块允许多个线程共享同一个已编译的正则表达式对象，也支持命名子组。

Re 模块: 核心函数和方法

正则表达对象: regex object
正则匹配对象: regex match object

核心提示: 我们发现，调用一个代码对象而不是一个字符串，性能上会有明显的提升。使用预编译的代码对象比直接使用字符串要快，因为解释器在执行字符串形式的代码前，都必须把字符串编译成代码对象。

由于正则表达式在执行过程中将进行多次比较操作，所以，预编译是必要的。

re.compile() 提供预编译功能。
purge() 函数能够用于清除这些缓存。

使用 compile() 函数编译正则表达式

匹配对象以及 group() 和 groups() 方法

处理正则表达式时，除了正则表达式对象之外，还有另一个对象类型: 匹配对象。
匹配对象: 成功调用 match() 或 search() 返回的对象。
匹配对象有两个方法: group() 和 groups().

group() 要么返回整个匹配对象，要么根据要求返回特定子组。、
groups() 仅返回一个包含唯一或者全部子组的元祖。如果没有子组的要求，那么当group()仍然返回整个匹配时，groups()返回一个空元组。

Python regex 也允许命名匹配

使用 match() 方法匹配字符串

match() 函数试图从字符串的 起始部分 对模式进行匹配。如果匹配成功，就返回一个匹配对象；如果匹配失败，就返回 None。匹配对象的 group() 方法能够用于显示那个成功的匹配。

由于match() 匹配不到会返回 None，所以匹配失败时调用 group() 方法，系统会抛出 AttributeError 的异常。

使用 search() 在一个字符串中查找模式()搜索与匹配的对比

search() 函数试图在字符串的 任意位置 对给定正则表达式模式搜索第一次出现的匹配情况。如果搜索到成功的匹配，就会返回一个匹配对象；否则，返回None.

m = re.search('foo', 'oo23foo45foo66')
if m is not None:
    print m.group()
结果是: foo

搜索成功，匹配失败。

重复、特殊字符以及分组

输入：

m = re.match("(a(b))", "ab")
if m is not None:
    print m.group()
    print m.group(1)
    print m.group(2)
    print m.groups()

输出：

ab
ab
b
('ab', 'b')

匹配字符串的起始和结尾以及单词边界

符号(^) 表示匹配字符串起始部分

使用 findall() 和 finditer() 查找每一次出现的位置

findall() 查询字符串中某个正则表达式全部的非重复出现情况。

非重复 示例：

m = re.findall("cc", "ccc")
if m is not None:
    print m

n = re.findall("cc", "cccc")
if n is not None:
    print n

输出：

['cc']
['cc', 'cc']

对于一个成功的匹配，每个子组匹配是由 findall() 返回的结果列表中的单一元素；对于多个成功的匹配，每个子组匹配是返回的一个元组中的单一元素，而且每个元组(每个元组都对应一个成功的匹配)是结果列表中的元素。

应用场景： 匹配一个完整电话号码的一部分(例如区号)，完整电子邮件地址的一部分(登录名称)

finditer(): 与 findall() 类似，但更节省内存。
例如：

s = 'this and that'
print re.findall(r'(th\w+) and (th\w+)', s, re.I)
print re.finditer(r'(th\w+) and (th\w+)', s, re.I).next().group()
print re.finditer(r'(th\w+) and (th\w+)', s, re.I).next().group(1)
print re.finditer(r'(th\w+) and (th\w+)', s, re.I).next().group(2)
print [g.groups() for g in re.finditer(r'(th\w+) and (th\w+)', s, re.I)]

输出：

[('this', 'that')]
this and that
this
that
[('this', 'that')]

注意： 使用 finditer 函数完成的所有额外工作都旨在获取它的输出来匹配 findall() 的输出。

使用 sub() 和 subn() 搜索和替换

输入: 切记范围中不能带 空格

m = re.sub("X", "Mr. Smith", "attn: X\n\nDear X,\n")
n = re.subn("X", "Mr. Smith", "attn: X\n\nDear X,\n")
o = re.sub(r'(\d{1,2})/(\d{1,2})/(\d{2}|\d{4})', r'\2/\1/\3', '2/20/91')

print m
print n
print o

输出：

attn: Mr. Smith
Dear Mr. Smith,
('attn: Mr. Smith\n\nDear Mr. Smith,\n', 2)
20/2/91

在限定模式上使用 split() 分隔字符串

扩展符号

输入:

m = re.findall(r"(?i)yes", "yes Yes YES good")
print m

输出：

['yes', 'Yes', 'YES']

(?:…) 不保存的匹配

输入：

m = re.findall(r"http://(?:\w+\.)*(\w+\.com)", "http://google.com http://www.google.com http://code.google.com")
print m

输出：

['google.com', 'google.com', 'google.com']

另外的例子： 

n = re.search(r"\((?P<areacode>\d{3})\) (?P<prefix>\d{3})-(?:\d{4})", "(800) 555-1212").groupdict()
print n

输出：

{'areacode': '800', 'prefix': '555'}

用子组索引替换：

m = re.subn(r"\((?P<areacode>\d{3})\) (?P<prefix>\d{3})-(?:\d{4})", "(\g<areacode>) \g<prefix>-xxxx", "(800) 555-1212")
print m

输出：

('(800) 555-xxxx', 1)

关于符号(?P=name):

m = bool(re.match(r"\((?P<areacode>\d{3})\) (?P<prefix>\d{3})-(\d{4}) (?P=areacode)-(?P=areacode)", "(800) 555-1212 800-800"))

n = bool(re.match(r"\((?P<areacode>\d{3})\) (?P<prefix>\d{3})-(\d{4}) (?P=areacode)-(?P=areacode)", "(800) 555-1212 800-900"))
print "m", m
print "n", n

输出：

m True
n False

(?(1)y|x) 例子：

m = re.search(r"(?:(x)|y)(?(1)y|x)", "xy")
print m.group()

输出：

xy
