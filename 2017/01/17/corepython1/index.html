摘要: 正则表达式

声明: 本篇博文为本人 python 核心编程的读书笔记

正则表达式 regex / RegEx

正则表达式为高级的文本模式匹配、抽取、与/或文本形式的搜索和替换功能提供了基础。
regex 是一些由字符和特殊符号组成的字符串，他们描述了模式的重复或者表述多个字符，于是 regex 能按照某种模式匹配一系列有相似特征的 字符串。

python中通过标准库中的 re 模块来支持正则表达式。

术语: 匹配 – 模式匹配(Pattern-Matching)

python中，主要通过两种方式实现 pattern-matching

  • 搜索 (searching)
  • 匹配 (matching)

元字符 – 赋予了正则表达式强大的功能和灵活性。

literal ['lɪt(ə)r(ə)l] 文字的，逐字的，无夸张的，字面，照字面的

使用择一匹配符号匹配多个正则表达式模式

| : 从多个模式中选择其一

符号 | 的出现增强了正则表达式的灵活性，使得正则表达式能够匹配多个字符串而不仅仅是一个。

择一匹配有时候也称作 并(union)或者 逻辑或(logical OR)

logical ['lɒdʒɪk(ə)l] 合逻辑的，合理的，逻辑学的，条理分明的，顺理成章，逻辑型

注: 需要显式匹配一个特殊符号或字符本身，必须使用反斜线转义功能。 “\”

创建字符集

方括号 [] 仅仅具有 逻辑或 的作用。对于单个字符的正则表达式，使用择一表达式和字符集是等效的。

限定范围和否定

字符集 [] 方括号中的两个符号中间用连字符 (-) 连接，用于指定一个字符的范围。
特殊的，如果脱字符 (^) 紧跟在左方括号后面，这个符号就表示不匹配给定字符集中的任何一个字符。

正常的，符号(^) 表示匹配字符串起始部分
example: [‘’-a]

使用闭包操作符实现存在性和频书匹配

星号或者星号操作符 (*) 将匹配其左边的正则表达式出现零次或者多次的情况(该操作称为 Kleene 闭包)。

加号操作符 (+) 将匹配一次或者多次出现的正则表达式(也叫做正闭包操作符)。

问号操作符 (?) 将匹配零次或者一次出现的正则表达式。

大括号操作符 ({}) 里面或者是单个值或者是一对由逗号分隔的值。这将最终精确地匹配前面的正则表达式N次或者一定范围的次数。

使用 问号(重载) 意味着，要么匹配0次，要么匹配1次，或者其他含义: 如果问号紧跟在任何使用闭合操作符的匹配后面，他将直接要求正则表达式引擎匹配尽可能少的次数。

Kleene 克林(人名：数学家)
closure ['kləʊʒə] 关闭，终止，结束，使终止，闭包

尽可能少的次数: 当模式匹配使用分组操作符时，正则表达式引擎试图“吸收”匹配该模式的尽可能多的字符–贪婪匹配; 问号要求正则表达式引擎去“偷懒”, 当前的正则表达式尽可能少的匹配字符，留下尽可能多的字符给后面的模式(如果存在)

表示字符集和特殊字符

与使用“0-9”这个范围表示十进制数相比，我们可以简单的使用 d 表示匹配任何十进制数字。另一个特殊字符(\w)能够用于表示全部字母数字的字符集，相当于[A-Za-z0-9_]的缩写形式，\s 可以用来表示空格字符。

注意：这些特殊字符的大写版本表示不匹配。

使用圆括号指定分组

使用正则表达式时，一对圆括号可以实现以下任意一个(或者两个)功能:

  • 对正则表达式进行分组
  • 匹配分组

圆括号的一个额外的作用: 匹配模式的字符串可以保存起来供后续使用。

扩展表示法

(?……) 扩展表示法以问号开始，通常用于在判断匹配之前提供标记，实现一个前视(或者后视)匹配，或者条件检查。

形式: (? P < name >)
