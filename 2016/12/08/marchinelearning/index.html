Machine Learn Introduction (Part One)

Headline: Introduction to Machine Learning: (Chapter one)

摘要: 机器学习导论 （第一章） By:Ehtem Alpaydm

声明: 本篇博文为本人阅读机器学习导论的读书笔记

为什么选择看着本书？

因为之前没有接触过算法以及机器学习的任何东西，所以就先从这本书看起

简介：

机器学习导论

目标:

对计算机编程，以便使用样本数据或者以往的经验来解决给定的问题。

词汇

拟合：

主要内容：

监督学习， 贝叶斯决策理论， 参数方法， 多元方法，维度规约， 聚类， 非参数方法， 决策树， 线性判别式， 多层感知器， 局部模型， 隐马尔可夫模型， 分类算法评估和比较， 组合多学习器以及增强学习等。

流程：

从已知实例中自动发现规律，建立对未知示例的预测模型；根据经验不断提高，不断改进预测功能以优化性能标准。

前言：

当人们不能直接编写计算机程序来解决给定的问题，而是需要借助实例数据或者经验时，就需要机器来学习。

需要学习的情况主要分为：

  • 当人们没有专门技术，或者不能解释他们的专门技术。
  • 要解决的问题随时间而变化或者依赖于特定的环境

第一章 绪论

第一节 什么是机器学习？

通过对数据实例或经验的分析，来构造一个好的或者有用的近似–模型，以达到解释数据形成过程的某些部分，帮我们了解过程，并预测过程。

机器学习的方法在大型数据库中的应用称为数据挖掘（data mining）

机器学习的过程是不断优化模型的过程，利用训练数据实例或者经验来优化模型的参数。

模型的作用：

预测性的：用于未来的预测
描述性的：从数据中获取知识
二者兼备

机器学习应用事例 ：

学习关联性：

关联规则（association rule）：

P (Y | X): 购买 X 的用户中，购买 Y 的概率
P (Y | X , D): 在满足 D 条件的人群里，购买 X 的用户中，购买 Y 的概率

这就是分域的思想，针对不同的域，提高数据精度，优化模型

分类 classification

创建一个分类器，将分类器获得的输入指派到其中的一个分类作为输出。

example:

if

condition1 > θ1 and condition2 > θ2 and …
then
category1
else if
conditionn….
then
category2
…
…
else
categoryn

我们利用拟合过去数据的规则，如果未来的数据与过去类似，那么，我们就可以用拟合的规则对新的实例来作出规则内正确的预测。

用关联性解释分类

针对只有两个分类: 机器语言表述为 1 / 0
P (Y | X):
我们把 X 看作某一个域（类似于 D） – 将这个域抽象为某一个属性
我们把 Y 看作数据的一个分类 – 将分类抽象为属性
公式解释为: 在满足 X 属性的条件下， 进入分类属性 Y 的概率

如果 P (Y | X) = 0.8
解释: 在 X 域这个属性的情况下， 属于分类 Y 的概率

注： 提取实例中的共同的特征进行分类，通过分类来捕获实例的规律, 继而优化特征，捕获更加好的且有用的近似规律 – 规则。

知识抽取
从数据中发现规律提取规则 称为 知识抽取。

规则
一种解释数据的简单模型，可以展示出 潜在数据处理 的解释。

回归

回归和分类均为监督学习的问题，及输入到输出的映射。

简单的拟合函数形式: 输入 X – 输出 Y, Wn 作为参数

线性

  • Y = W * X + Wo
    二次函数
  • Y = W2 X X + W1 * X + W0
    …
    …
    高阶多项式
    其他非线性函数

Example:
依赖与一组参数的模型：

Y = g (X | θ)
g (*) 是模型， θ 是模型的参数

对于分类，Y 是类编码（1 / 0），g (*) 为 类判别式
对于回归， Y 是数值，g (*) 为 回归函数

响应面设计 – response surface design

在回归模型中选择最优的样本，根据靠近最有样本的最新实例，加入训练数据，优化参数，拟合新的模型。

非监督式学习

当我们只有输入数据，没有任何前置的条件判断，而要通过对输入数据的观察，自己来发现输入数据隐藏的规律以及某种结构，使特定的模式比其他模式更常出现，使我们知道，在某些条件下，哪些经常发生，哪些不经常发生，在统计学中也叫 密度估计

密度估计的一种方法叫做 聚类，目标是发现输入数据的簇或者分组

聚类的分组可以用来识别 离群点

  • 发现异常或者筛选错误数据
  • 发现新的市场或者新的机会
增强学习

在某些应用情况中，系统的输出是一系列动作的序列，这个序列成为策略。
我们只在策略的层面上评判，如果策略是好的策略，那么这个策略中的一系列动作我们都认为是好的动作，这个策略也称为 达到目标的正确动作的序列

能够评估策略的好坏程度，并从以往好的动作序列 (即策略) 中学习，以便能够产生策略的这种方法叫做增强学习 (reinforcement learin) 算法

综述

看到这一节，书籍翻译是注释，可是我觉得不是，所以我根据自己的理解起名为综述
这一节，第一段话我觉得很赞，就摘录修改如下

进化是形成我们的身体形状和我们内在本能的主要力量, 他赋予我们大脑和学习机制，使我们可以根据经验实现自我更新。我们还需要终生学习，以改变我们的行为，从而适应包括进化论还不能预测的和可预测的各种环境。

几乎所有的科学领域都在用模型拟合数据。科学家设计实验、进行观测并收集数据。然后，通过寻找解释所观测数据的简单模型，尝试抽取知识。该过程称为归纳（induction），它是从一组特别的示例中提取通用规则的过程。
